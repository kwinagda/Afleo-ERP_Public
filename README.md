# Afleo ERP Public

This repository is the public-share package for the Afleo Trade Intelligence Console.

Public files included:
- `index.html`
- `afleo_dgft_v2.html`

Purpose:
- provide a direct employee-access entry point
- keep internal source workbooks and internal build documents out of the public repository

Recommended deployment:
- enable GitHub Pages on this repository
- serve from `main` branch, root folder

Expected public URL:
- `https://kwinagda.github.io/Afleo-ERP_Public/`

This repo should contain only the deployable static files, not the internal source Excel files or internal project logic documents.
